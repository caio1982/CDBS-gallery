Source code

The shell script and the resources files are available in my personal Github repository and you can fetch them by running the following command:

git clone https://github.com/caio1982/CDBS-gallery.git/

The poor documentation inside the script can guide you through the steps necessary to "mirror" this online gallery. If you only want to check how it does look like, just click on the link above and browse the online tree.

Any question or doubts you may have concerning the source code, please take a look at the FAQ first. A lot of questions already were answered there :-)

Introduction

NEW! now you can browse the source packages by the CDBS includes used.

This pet project aims to create and mantain (we hope so) a repository with all the CDBS rules, that builds all binary packages, used across the whole Debian archive. It can, and in fact that's intentional, serve as a guide for beginners trying to figure out how CDBS works.

There are currently MIN_OUTPUT source packages using the CDBS framework in the Debian archive at the moment (which corresponds to ~CUT_OUTPUT% of MAX_OUTPUT source packages).
